HTML and CSS

The base technologies of Web Development

Posted by Anton Talalaev on November 23, 2020

Nowadays, the World Wide Web is the main force that influences our lifestyle and all humanity itself. In this article, in simple words, I would like to touch two main technologies based on which our browsers can show us the content of web pages.

First of all, you need to know that any browser, basically, is a virtual machine that can draw images based on appropriate instructions. The main (and also the base) technologies for such instructions are HTML and CSS.

Hypertext Markup Language (HTML) is the language for describing what the content will be found on the page and what will be the basic structure of the content.

On the other hand, Cascading Style Sheets (CSS) is the language for describing all the styles that will be applied to the content that was described using HTML.

As you can see, both technologies, HTML and CSS, are always used together to deliver nice-looking web pages to users.

As I said before, HTML and CSS provide web developers the way to write instructions for browsers. What does it mean?

How does it all works - HTML

So, let’s look at common HTML instruction:

<h1> Here will be the header text of the page </h1>

All HTML instructions are based on tags. In this case, you can see the header tag - <h1>. This tag is used to show the header text of the page.

Each HTML tag has three main parts:

  • opening tag - <h1>.
  • content - “Here will be the header text of the page”.
  • closing tag - </h1>.

Examples of frequently used HTML tags:

  • <h1> - header tags (<h1>, <h2>, <h3>, <h4>, <h5>, <h6>).
  • <p> - paragraph tag.
  • <a> - anchor tag, is used to provide links to other websites and resources.
  • <ol> / <ul> - ordered list and unordered list.
  • <img> - image tag, to set image to the page.
  • <div> - division tag, to union other tags in blocks.

The simple HTML-document will look like this:

<!DOCTYPE html>

<html lang="en">

<head>

<meta charset="UTF-8">

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<title>Document</title>

</head>

<body>

<h1>Here we will find the main topic of the page</h1>

<section class="article">

<h2>The name of the First article should be here</h1>

<p>Here will be paragraph 1</p>

<p>Here will be paragraph 2</p>

<p>Here will be paragraph 3</p>

</section>

<section class="article">

<h2>The name of the Second article should be here</h1>

<p>Here will be paragraph 1</p>

<p>Here will be paragraph 2</p>

<p>Here will be paragraph 3</p>

<p>Here will be paragraph 4</p>

</section>

</body>

</html>

All the tags are used to mark the content inside it. If the content is marked well, we have the opportunity to use CSS to style it.

How does it all works - CSS

Now let’s look at common CSS instruction:

h1 {

background-color: purple;

font-family: 'Times New Roman';

}

All CSS instructions are based on selectors. In this case, as a selector, you can see the HTML header tag - h1. This selector shows us coverage of styles above - the instruction will cover all <h1> tags of the HTML document.

Each CSS instruction has three parts:

  • selector - h1 (describe which HTML tags will be covered by CSS style).
  • declaration block - the piece of code surrounded by curly braces.
  • declarations - background-color: purple (CSS styles itself).

Let's look at a bit more examples of CSS instructions:

h1 {

background-color: purple;

font-family: 'Times New Roman';

}

section {

background-color: aquamarine;

font-family: 'Arial';

border: 5px solid black;

text-align: center;

}

p {

font-style: italic;

}

h2 {

background-color: lightgreen;

}

As a conclusion

HTML and CSS both are the basis for the entire modern web content delivery. These technologies are used together to provide users with full of content and nice-looking web pages.
